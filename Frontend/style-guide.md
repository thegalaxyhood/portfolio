## Portfolio style guide

## Layout

The designs were created to the following widths:

- Mobile: 400px
- Tablet: 900px
- Desktop: 1240px

## Colors

### Primary

- Light Green:#00cdac
- Dark Green: #02aab0
- Header Text Color: #272341
- Body Text Color: #222

### Neutral

- Neutral Light: #fff
- Neutral Dark: #333333

## Typography

- Family: [Montserrat](https://fonts.google.com/specimen/Montserrat)

- Weights: 300, 400, 700
